# Interface: ModuleDefinition

禅道模块定义，由多个动作组成。

## Properties

| Property | Type | Description |
| ------ | ------ | ------ |
| <a id="property-actions"></a> `actions` | [`ModuleAction`](ModuleAction.md)[] | 模块支持的动作集合。 |
| <a id="property-description"></a> `description?` | `string` | 模块说明。 |
| <a id="property-display"></a> `display?` | `string` | 面向用户展示的模块名称。 |
| <a id="property-name"></a> `name` | [`ModuleName`](../type-aliases/ModuleName.md) | 模块名称，例如 `product`、`bug`。 |
